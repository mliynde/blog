金属间的置换反应

初中阶段使用的金属活动性顺序表：

置换反应

我们知道，一种金属单质能与另一种活动性更弱的金属的盐溶液发生反应。这种反应属于置换反应致幻反应，是氧化还原反应的一种。

金属单质作为还原剂，发生氧化反应（价态升高）；盐作为氧化剂，发生还原反应。

举例：

  • $\mathrm{Fe+CuSO_4 = Cu+FeSO_4}$
  • $\mathrm{3Mg+2AlCl_3=2Al+3MgCl_2}$
  • $\mathrm{Cu+2AgNO_3=2Ag+Cu(NO_3)_2}$

上面说的是一种金属单质与一种盐溶液发生反应，那么……

单金属多盐

下面是一个一金属单质多盐的例子：

将一定质量的 $\mathrm{Zn}$ 粉加入 $\mathrm{Mg(NO_3)_2}$、$\mathrm{Fe(NO_3)_2}$、$\mathrm{Cu(NO_3)_2}$、$\mathrm{AgNO_3}$ 这四种盐的水溶液中，充分反应后过滤，得到滤液和滤渣：

  • 向滤液中加入稀盐酸，发现有白色沉淀产生，则滤液中一定有 $\mathrm{Mg(NO_3)_2}$、$\mathrm{Zn(NO_3)_2}$、$\mathrm{Fe(NO_3)_2}$、$\mathrm{Cu(NO_3)_2}$、$\mathrm{AgNO_3}$ 。
  • 向滤渣中加入稀盐酸，发现有无色气泡产生，则滤渣中一定有 $\mathrm{Ag}$，$\mathrm{Cu}$，$\mathrm{Fe}$，可能有 $\mathrm{Zn}$，不可能有 $\mathrm{Mg}$。

现在要探究反应过程。

怎样反应？

然而，单独把 $\mathrm{Zn}$ 粉加入一种盐溶液，都能发生反应（活动性排在 $\mathrm{Zn}$ 之前的除外）。那么最终的反应结果会是什么样的？

假设只加入那么一点点 $\mathrm{Zn}$ 粉，与 $\mathrm{Zn}$ 相关的化学方程式有：

  • $\mathrm{Zn+Fe(NO_3)_2 = Fe+Zn(NO_3)_2}$，①
  • $\mathrm{Zn+Cu(NO_3)_2 = Cu+Zn(NO_3)_2}$，②
  • $\mathrm{Zn+2AgNO_3 = 2Ag+Zn(NO_3)_2}$，③

注意：方程式①和②生成的 $\mathrm{Fe}$ 和 $\mathrm{Cu}$ 是能接着与后面的盐溶液发生反应的，而 $\mathrm{Ag}$ 不会！那么，如果把所有可能的化学方程式（包括所有金属单质和盐溶液）写在一起，会是什么样的呢？

Zn Fe Cu Ag
Zn盐 — — — —
Fe盐 $\mathrm{Zn+Fe(NO_3)_2\\=Fe+Zn(NO_3)_2}$ — — —
Cu盐 $\mathrm{Zn+Cu(NO_3)_2\\ = Cu+Zn(NO_3)_2}$ $\mathrm{Fe+Cu(NO_3)_2\\ = Cu+Fe(NO_3)_2}$ — —
Ag盐 $\mathrm{Zn+2AgNO_3\\ = 2Ag+Zn(NO_3)_2}$ $\mathrm{Fe+2AgNO_3\\ = 2Ag+Fe(NO_3)_2}$ $\mathrm{Cu+2AgNO_3\\ = 2Ag+Cu(NO_3)_2}$ —

每种反应生成的金属单质代回表格，如果能反应，那么刚生成的金属单质又会被消耗掉，生成活动性更弱的金属，反映在表格里，越来越趋向右下角。

同理，每种反应生成的盐代回表格，如果能反应，那么刚生成的盐又会被消耗掉，生成活动性更强的金属盐，反映在表格里，越来越趋向左上角。

总而言之，最终的反应结果等价于：

  • 活动性最强的金属单质 与 活动性最弱的金属盐溶液 发生置换反应，
  • 生成 活动性最弱的金属 和 活动性最强的金属对应盐溶液。
  • 方程式就是表格里最左下角的化学方程式：$\mathrm{Zn+2AgNO_3 = 2Ag+Zn(NO_3)_2}$。
  • 简记：（最）强金（最）弱盐 反应物，（最）弱金（最）强盐 生成物。

如果盐溶液完全反应，只需要把该行去掉，剩下的反应依然遵循这一规则。

反应过程图

现在要探究反应过程，以加入 $\mathrm{Zn}$ 粉的质量为横轴，横轴上方对应滤渣，下方对应溶质。

直接上反应全过程的图（为方便起见，图片中横轴是弯折的）：

颜色说明

  • 黑色：原来就存在于溶液中，且滤渣中没有对应单质。
  • 深灰色（不明显）：滤液中仍有此盐，且滤渣中没有对应盐。
  • 浅灰色：滤液中没有此盐，对应金属全部成为滤渣（完全反应）。
  • 浅橙色：滤渣中已经有此金属单质，且溶液中溶解着对应盐。
  • 橙色：
    • 滤渣中已经有此金属单质，但溶液中没有对应盐（完全反应）。
    • 此金属盐是新生成的。

根据检验结果，找寻反应全过程中对应的内容，分析成分……

发现：只要发生反应，滤液中就一定有 $\mathrm{Zn}$ 盐，滤渣中就一定有 $\mathrm{Ag}$ 单质；$\mathrm{Mg}$ 盐自始至终都不参与反应。

多金属多盐

这个又该怎么分析？

将一定质量的 $\mathrm{Zn}$ 粉和 $\mathrm{Fe}$ 粉均匀混合后，加入$\mathrm{Cu(NO_3)_2}$、$\mathrm{AgNO_3}$ 这两种盐的水溶液中，充分反应后过滤，得到滤液和滤渣。

  • 向滤液中加入稀盐酸，发现有白色沉淀产生，则滤液中一定有 $\mathrm{Zn(NO_3)_2}$、$\mathrm{Fe(NO_3)_2}$、$\mathrm{Cu(NO_3)_2}$、$\mathrm{AgNO_3}$ 。
  • 向滤渣中加入稀盐酸，发现有无色气泡产生，则滤渣中一定有 $\mathrm{Ag}$，$\mathrm{Cu}$，$\mathrm{Fe}$，可能有 $\mathrm{Zn}$。

依然要探究反应过程。实际上，这种反应依然遵守上表的规则：（最）强金（最）弱盐反应物……。

反应结果表（轴）

画一个两行的表格，按活动性越来越弱的顺序，把反应前存在的金属单质从左往右依次写在第一行，盐写在第二行，像这样：

金属 $\mathrm{Zn}$ $\mathrm{Fe}$
盐 $\mathrm{Cu(NO_3)_2}$ $\mathrm{AgNO_3}$

以「产生白色沉淀」为例：

  1. 确定滤液中一定有 $\mathrm{AgNO_3}$，也就是说 $\mathrm{AgNO_3}$ 过量；

  2. 找到 $\mathrm{AgNO_3}$ 的位置，固定住；

  3. 模拟反应过程：

    ① 从左往右找第一个单质；

    ② 从右往左找第一个盐；

    ③ 两者反应，把生成物写在对应位置，并作上标记（刻意写斜或画圈）；

    ④ 由于$\mathrm{AgNO_3}$ 一定过量，在 $\mathrm{Zn}$ 上面打一个「×」号，表明被完全消耗；

    ⑤ 重复①~④，直到找不到金属单质。

  4. 反应结束，分析成分。

最终表格（不太明显）：
| 金属 | $\color{Gray}\mathrm{Zn}\times$ | $\color{Gray}\mathrm{Fe}\times$ | | $\color{Orange}\cal{Ag}$ |
| —-: | ——————- | —————- | —— | —— |
| 盐 | $\color{Orange}\cal{Zn(NO_3)_2}$ | $\color{Orange}\cal{Fe(NO_3)_2}$ | $\mathrm{Cu(NO_3)_2}$ | $\mathrm{AgNO_3}$ |
